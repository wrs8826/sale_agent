# 无锡高新区（新吴区）飞凤人才计划——资金政策

> 数据来源：《2025年无锡高新区（新吴区）"飞凤人才计划"科技领军人才创新创业项目申报公告》附件1、附件2，2025年9月26日印发。

---

## 一、领军人才团队支持（附件1）

### 创业领军人才团队

三年内给予项目资金（启动资金+风险投资奖励+贷款贴息）**不超过800万元**，一般不超过项目实际投入资金。

| 支持项目 | 内容 |
|---|---|
| **启动资金** | 最高500万元，其中10%用于补助人才；原则上三年拨付，按5:5支付 |
| **风险投资奖励** | 按备案创投基金或风险投资机构实际到位投资额的15%给予奖励，最高200万元 |
| **贷款贴息** | 按银行同期LPR利率计算贷款利息的50%补贴，补贴总额≤100万元 |
| **间接融资支持** | 人才贷、苏科贷、锡科贷、新科贷等省市区风险补偿贷款，最高**1000万元** |
| **创业场地** | 根据项目实际需求提供办公启动场所，给予三年房租补贴 |
| **人才公寓** | 提供团队公寓住房**3套**，三年内免收租金 |

### 创新领军人才团队

三年内给予项目资金**不超过500万元**，一般不超过项目实际投入资金。

| 支持项目 | 内容 |
|---|---|
| **启动资金** | 最高500万元，其中10%用于补助人才，并不得抵扣工资待遇；原则上三年拨付，按5:5支付 |
| **人才公寓** | 提供团队公寓住房**3套**，三年内免收租金 |

---

## 二、创新创业领军人才支持（个人，附件2）

### 创业领军人才（个人）

三年内给予项目资金（启动资金+风险投资奖励+贷款贴息）**不超过500万元**，不超过项目实际投入资金。

| 支持项目 | 内容 |
|---|---|
| **启动资金** | 最高300万元，其中10%用于补贴人才；原则上三年拨付，按5:5支付 |
| **风险投资奖励** | 按备案创投基金或风险投资机构实际到位投资额的15%给予奖励，最高100万元 |
| **贷款贴息** | 按银行同期LPR利率计算贷款利息的50%补贴，补贴总额≤100万元 |
| **间接融资支持** | 人才贷、苏科贷、锡科贷、新科贷等省市区风险补偿贷款，最高**500万元** |
| **创业场地** | 根据项目实际需求提供办公启动场所，给予三年房租补贴 |
| **人才公寓** | 提供领军人才公寓住房**1套**，三年内免收租金 |

### 创新领军人才（个人）

三年内给予项目资金**不超过100万元**，不超过项目实际投入资金。

| 支持项目 | 内容 |
|---|---|
| **启动资金** | 最高100万元，其中≥30%用于补助人才，不得抵扣工资待遇；原则上三年拨付，按5:5支付 |
| **人才公寓** | 提供领军人才公寓住房**1套**，三年内免收租金 |
