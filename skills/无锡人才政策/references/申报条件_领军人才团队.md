# 无锡高新区（新吴区）飞凤人才计划——领军人才团队申报条件（附件1）

## 通用基本条件

1. **团队规模：** 1名团队带头人 + 2-5名核心成员；核心成员专业结构合理，具有较强关联性和互补性。
2. **工作经历：** 团队带头人及核心成员有5年以上（具有博士学位的有2年以上）在国内外知名企业、高校、科研单位从事研发或管理等工作经历。
3. **在锡工作承诺：** 入选后须在申报企业继续工作3年以上；带头人每年累计在锡工作≥1个月并从申报单位缴纳个税；核心成员每年在锡工作≥6个月并缴纳个税或社保。
4. **青年人才占比：** 团队中40周岁以下（1985年1月1日后出生）的青年人才比例不低于30%。
5. **核心成员学历：** 一般应具有硕士学位或副高级职称（软件互联网、服务外包领域可适当放宽）；境外学历需提供教育部留学服务中心认证报告。

## 带头人资格（具备以下之一）

1. 中国科学院院士、中国工程院院士
2. 主要科技发达国家科学院、工程院院士（不含通讯院士、外籍院士）或其他海外杰出人才
3. 国际重要科技奖项获得者
4. 省部级科技进步奖一等奖以上，或国家科技进步奖二等奖以上主要完成人
5. 省级以上产业类重大研发项目或科技类重大项目主要承担人（列前3名）
6. 国家重大人才工程入选者、长江学者、国家杰出青年/优秀青年科学基金项目负责人、中科院"百人计划"入选者等
7. 江苏省"双创计划"入选者或外省相当层次称号获得者
8. IEEE Fellow、IEE Fellow、IEC、APS、AIMBE、ACM、ASME、SIAM、AIAA、RSC等国际著名学术组织高级成员（Fellow）
9. 上海交大/泰晤士报全球排名前100名境内外大学的教授、副教授（限申报时最新排名）
10. 省级以上制造业创新中心、企业技术中心及其他同层级企业创新平台负责人
11. 其他相当于上述层次的人才

## 创业领军人才团队（分类条件）

1. **来锡时间：** 团队应于2022年1月1日以后引进到无锡，并在新吴区依法注册成立具有独立法人资格的企业
2. **带头人持股及出资：** 带头人应为企业主要创办人且为第一大股东或最大自然人股东，个人到位货币出资≥300万元；通过持股公司出资的折算货币出资≥300万元
3. **带头人职务：** 应为申报企业主要负责人（担任董事长、经理等职务，已在工商信息系统备案）；核心成员在申报单位全职工作担任研发机构主要负责人或关键研发项目主持人及以上职务
4. **申报单位资质（2022年以来，具备之一）：** ①国家或省认定的单项冠军/专精特新/高新技术企业等；②拥有省级以上重点实验室或企业创新平台；③省级主管部门单项300万元以上财政资助；④获得国家专项资金单项500万元以上财政资助；⑤团队带头人领衔的团队获得引才计划单项300万元以上财政资助（截至申报截止前实际拨付≥50万元）
5. **核心成员薪酬：** 从到锡工作次月起，申报单位给予核心成员月均薪酬≥1万元，合同期内不得减少
6. **外籍代持：** 外籍人才可由配偶代持；若配偶同为外籍则可由中国籍父母或子女代持

## 创新领军人才团队（分类条件）

1. **来锡时间：** 团队成员均于2022年1月1日以后引进到无锡，已与申报单位签订正式聘用合同
2. **薪酬：** 带头人月均薪酬≥4万元，核心成员月均薪酬≥2万元，合同期内不得降低
3. **研发投入：** 申报单位对项目研发投入≥300万元（截至申报截止日期实际投入≥100万元）；且自2022年以来获得下列项目支持之一：①国家单项500万元以上财政无偿资助；②江苏省部级单项300万元以上无偿资助；③团队带头人领衔团队获引才单项50万元以上财政无偿资助（截至截止日期已拨付≥25万元）
4. **申报单位为企业的额外条件：** 须于2024年12月31日前在新吴区完成工商注册，并满足：①国家或江苏省/无锡市人才工程入选对象创办；②国家或省认定的高新技术企业/独角兽/瞪羚等；③拥有省级及以上企业创新平台；④获得省级及以上部门或市、县（区）引才计划资助≥30万元
5. **合同期内配套：** 合同期内申报单位配套用于引进人才科研和成果转化的自筹经费（不含财政资金）≥200万元
