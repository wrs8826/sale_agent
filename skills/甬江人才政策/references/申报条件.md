# 2026 甬江人才工程——支持领域与申报条件

## 支持领域

重点引进支持三类项目：

- **制造业项目**：绿色石化、新能源汽车及高端装备、新材料、人工智能与机器人、新一代信息技术、绿色能源、生命健康、时尚消费品、航空航天与海工装备等，助力打造未来制造、未来信息、未来材料等前瞻性未来产业
- **高校院所科研项目**：面向宁波高水平大学建设和关键核心技术攻关，支持具有较强原始创新能力的人才和团队
- **现代服务业和现代农业项目**：电子商务、港航物流、金融保险、教育文体、专业服务、规划设计、时尚创意、科技服务；现代种业、智慧农业与装备、农业绿色发展、农产品加工与质量安全、乡村运营与产业融合等

同等条件下优先支持：3 年以上海外经历的人才领衔项目、已落地宁波实际经营的创业项目、已获专业机构投资的项目、已获区（县、市）或开发区支持的项目。

## 一、制造业项目申报条件

仅限依托在甬企业申报，共 5 个类别。"校（院）企双聘"人才可依托所在企业申报。

### 1. 创业人才

1. 学历：硕士及以上；从业经历：3 年以上（截至 2026-06-15 满 3 年）；年龄：不超过 60 周岁（1966-01-01 后出生）
2. 计划来宁波创业，或来甬创业（工作）不超过 3 年（2023-01-01 以后）
3. 须为企业创始人、法定代表人、第一大股东（不含股权穿透）或最大自然人股东；1 家企业只能申报 1 个创业人才项目
4. 入选后企业须为主导运营实体（非子公司/分公司）；申报人实际到位货币出资 ≥ 100 万元

### 2. 创新人才

1. 学历：博士；从业经历：3 年以上；年龄：不超过 60 周岁
2. 近 2 年内来甬或计划来甬（2024-01-01 以后），依托宁波企业开展创新成果转化或基础研究；入选后须全职在甬连续工作 5 年以上，支持期内不得在甬外单位全职工作
3. 依托企业成立 1 年以上，财务运行良好，能提供科研资金、研发设备及产业化所需要素
4. 依托企业法定代表人、实际控制人、主要创始人或最大自然人股东**不得**申报创新人才项目

### 3. 青年人才

学历：博士；从业经历：1 年以上（截至 2026-06-15 满 1 年）；年龄：不超过 40 周岁（1986-01-01 后出生）。其他条件与创新人才相同。

### 4. 创业团队

团队须拥有关键核心技术，有望提升宁波产业水平。

1. 带头人：硕士及以上，3 年以上从业经历，年龄 ≤ 65 周岁（1961-01-01 后出生）；成员：硕士及以上或本科+3 年经历，年龄 ≤ 55 周岁（1971-01-01 后出生）；团队至少 1 名带头人 + 4 名成员
2. 带头人和成员来甬 ≤ 3 年（2023-01-01 以后）；带头人须为创始人、法定代表人和第一大股东；带头人+成员合计持股 > 50%，持股成员 ≥ 2 名
3. 入选后企业须为主导运营实体；企业实际到位货币出资 ≥ 200 万元，带头人到位货币出资 ≥ 100 万元；支持期内过半成员须在项目企业全职工作

### 5. 资本引才团队

团队须自带技术、项目和资金，有较好市场前景。

1. 带头人：硕士及以上，3 年以上经历，年龄 ≤ 65 周岁；成员：硕士及以上或本科+3 年经历，年龄 ≤ 55 周岁；至少 1 名带头人 + 4 名成员
2. 带头人和成员来甬 ≤ 3 年，且已在宁波创办企业（截至 2026-06-15 完成工商注册）；带头人须为创始人、法定代表人和第一大股东；带头人+成员合计持股 > 40%，持股成员 ≥ 2 名
3. 企业实际到位货币出资 ≥ 500 万元；带头人到位货币出资 ≥ 100 万元；企业已获民间资本（排除政府性出资、团队自有、带头人直系亲属及配偶出资）实到 ≥ 300 万元（含资本公积），且须于 **2026-03-15 前**到账
4. 企业须为主导运营实体；支持期内过半成员须全职工作

## 二、高校院所科研项目申报条件

主要依托在甬高校、科研院所（含产业技术研究院、医疗卫生单位含民营医疗机构）申报，共 3 个类别。申报人研究方向一般应与单位重点建设学科或发展方向契合。

### 1. 创新人才

条件与制造业创新人才相同。

### 2. 青年人才

条件与制造业青年人才相同。

### 3. 创新团队

1. 带头人：博士，3 年以上全职经历，年龄 ≤ 55 周岁；成员：博士，或硕士+3 年全职经历，年龄 ≤ 45 周岁（1981-01-01 后出生）；至少 1 名带头人 + 4 名成员
2. 带头人和成员计划来甬，或来甬 ≤ 2 年（2024-01-01 以后）；入选后须全职在依托单位连续工作 5 年以上，支持期内不得在甬外单位全职工作
3. 依托单位成立 1 年以上，运行良好，能提供科研资金、设备及产业化要素
4. 依托单位法定代表人、实际控制人、主要创始人或最大自然人股东**不得**作为带头人或成员申报

## 三、现代服务业和现代农业项目申报条件

主要依托在甬企业申报（教育人才可依托中小学、中职学校），共 5 个类别。

### 1. 创业人才

申报人实际到位货币出资 ≥ 50 万元。其他条件与制造业创业人才相同。

### 2. 创新人才

学历：硕士及以上；从业经历：3 年以上。其他条件与制造业创新人才相同。

### 3. 青年人才

学历：硕士及以上；从业经历：1 年以上；年龄 ≤ 40 周岁。其他条件与制造业青年人才相同。

### 4. 创业团队

企业实际到位货币出资 ≥ 100 万元，带头人到位货币出资 ≥ 50 万元。其他条件与制造业创业团队相同。

### 5. 创新团队

带头人：硕士及以上，3 年以上全职经历；成员：硕士及以上或本科+3 年全职经历。其他条件与高校院所创新团队相同。
