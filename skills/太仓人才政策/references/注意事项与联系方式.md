# 太仓市科技领军人才计划——注意事项与联系方式

## 一、主要注意事项

1. **同一企业限制：** 同一家企业一般不可同时申报重大创新创业团队、创业领军人才、创新领军人才。
2. **重复资助限制：** 已在苏州所辖各市、区获得过人才引进资助的，原则上不予受理。
3. **申报次数：** 同一人才申报同类项目不超过2次；同一项目同一年度只能申报1次。
4. **破格申报：** 可在注册时间、工作年限、学历等条件上适当放宽，但数量不超过当年有效申报总数的5%，须由镇区推荐并提交破格推荐材料。
5. **材料真实性：** 申报材料须客观、真实、完整；弄虚作假等不良信用行为将取消资格、记入诚信档案、5年内不再受理。
6. **项目变更：** 项目实施过程中，项目负责人不得变更；研究内容和预期指标原则不予调整。
