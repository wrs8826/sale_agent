# 太仓市科技领军人才计划——重大创新创业团队申报条件

**适用人群：** 能突破产业关键核心技术、推动重大科技成果转化的顶尖团队。

| 条件 | 要求 |
|---|---|
| 团队规模 | 至少3人（领军人才+核心成员） |
| 学历 | 博士学位+2年以上知名企/高校/科研院所研发管理经历；省级及以上创业类人才领衔、5年以上工作经历的可放宽为硕士 |
| 创业团队出资 | 领军人才现金出资≥100万元；团队成员总股权≥30%，团队成员总现金出资≥200万元 |
| 创新团队薪酬 | 成员担任高管或关键研发项目负责人，税前年薪≥30万元 |
| 项目规模 | 总投入≥1亿元，已投入经费≥3000万元（创业类可为已获社会融资≥2000万元） |
| 来太时间 | 2023年1月1日后来太；创业团队领军人才为省级及以上创业类人才的可放宽为2021年1月1日后 |

**申报限制：** 同一企业不可同时申报重大创新创业团队、创业领军人才、创新领军人才。
