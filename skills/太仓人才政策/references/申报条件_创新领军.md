# 太仓市科技领军人才计划——创新领军人才申报条件

**适用人群：** 到太仓科技型企业领衔研发及管理，能促进企业转型升级的领军人才。

## 已落户项目基本条件

| 条件 | 要求 |
|---|---|
| 学历 | 博士学位+2年以上知名企/高校/科研院所研发管理经历；年薪60万元且工作经历5年以上的可放宽为硕士 |
| 引进时间 | 2024年1月1日后引进，2025年8月31日前正式到岗 |
| 职务 | 担任研发机构主要负责人、关键研发项目主持人及以上职务 |
| 薪酬 | 税前年薪≥30万元，合同期内不得降低 |
| 服务年限 | 引进后全职连续服务3年以上 |
| 引进单位资质（具备之一） | 上市企业/国家高新技术企业/专精特新"小巨人"；国家或省认定科技型企业；拥有省级及以上创新平台；独角兽/瞪羚企业；苏州市级及以上创业领军人才企业 |

**优先支持：** 40周岁以下青年人才、企业高薪聘用人才、海外引进人才。

## 未落户项目

满足已落户条件（1）和引进单位条件，另须：
- 截止2025年8月31日尚未来太全职到岗，来源为江苏省外
- 2026年6月30日前能全职到岗，连续服务3年以上
- 引进企业意向给予税前年薪≥60万元

## 申报限制

- 同一人才申报同类项目不超过2次，同一项目同一年度只能申报1次
- 已在苏州所辖各市、区获得过人才引进资助的，原则上不予受理
- 同一家企业一般不可同时申报重大创新创业团队、创业领军人才、创新领军人才
- 破格申报（放宽注册时间/工作年限/学历）：数量不超过当年有效申报总数的5%
