# 太仓市科技领军人才计划——创业领军人才申报条件

**适用人群：** 带团队、带技术、带项目来太创业，兼具科技与市场能力的领军人才。

## 通用基本条件

| 条件 | 要求 |
|---|---|
| 学历 | 全日制本科及以上 |
| 经历 | 3年以上知名企/高校/科研院所研发、管理经历或自主创业经历 |
| 企业职位 | 企业主要负责人（担任董事长、执行董事、总经理任一职务） |
| 持股 | 股权≥20% 或为自然人第一大股东（建议>30%） |
| 实缴现金出资 | ≥100万元（35周岁以下青年可放宽至50万元）；通过持股公司出资的折算≥200万元 |
| 员工社保 | 有2名及以上非股东员工缴纳社会保险费 |
| 外籍代持 | 内资公司的，股份可由直系亲属（夫妻、父母、子女）代持 |

## 来太时间要求（已落户项目）

- 一般：2023年1月1日至2025年8月31日之间来太创业并完成工商注册、验资
- 2026年第一批：2024年1月1日至2026年4月30日
- 2026年第二批：2024年1月1日至2026年8月31日

## 未落户项目

满足通用条件（1）-（3），在获得认定资格后于2026年6月30日前在太完成工商注册、验资等相关工作。

## 专项申报（择一）

| 专项 | 条件 |
|---|---|
| 重点产业专项 | 项目属于生物医药、航空航天、集成电路、人工智能产业领域 |
| 资本引才专项 | 近三年（2022年1月1日后）累计获得备案风投机构到账投资超500万元 |

**优先支持：** 40周岁以下青年人才、全职来太创业项目、沪上转移来太的优质项目。
