# 2026年成都AI产业专项补贴——模型使用补贴与企业融资奖励

## 一、模型使用补贴（最高100万元）

### 支持标准

对企业通过API调用、云化部署等方式使用模型的，按不超过年度模型使用费用的**30%**，给予最高**100万元**补贴。

### 申报条件

1. 使用的模型为经生成式人工智能服务备案、登记或通过大模型国家标准测试认证的人工智能模型。
2. 模型使用方式包括模型订阅、API调用、云化部署等。
3. 模型用途仅限于人工智能软硬件开发应用。

### 申报材料

1. 模型使用补贴项目申报表。
2. 注册登记证照（申报单位统一社会信用代码证复印件及法定代表人身份证复印件）。
3. 申报单位2025年度审计报告（会计师事务所出具，报告每页右下角含二维码）。
4. 申报单位2025年度使用模型的采购合同、发票、银行划款凭证等相关证明材料。
5. 模型经生成式AI服务备案、登记或通过大模型国家标准测试认证的证明材料。
6. 申报单位使用模型用于人工智能软硬件开发应用的证明材料。
7. 其他需要补充的有关证明材料。

**政策咨询：** 市经信局人工智能产业处，电话：028-61884670

## 二、企业融资奖励（最高100万元）

### 支持标准

对获得天使投资、风险投资和私募股权投资，且人工智能相关业务收入占比60%以上的人工智能企业，按不超过企业实际获得投资额的**10%**，给予最高**100万元**奖励。

### 申报条件

1. 申报主体2025年度主营业务中人工智能业务收入占比达到60%（含）以上。
2. 申报主体2025年度获得天使投资、风险投资和私募股权投资金额之一即可。

### 申报材料

1. 支持企业融资项目申报表。
2. 注册登记证照（统一社会信用代码证复印件及法定代表人身份证复印件）。
3. 申报单位2025年度审计报告。
4. 申报单位2025年度人工智能业务收入专项审计报告（会计师事务所出具，报告每页右下角带二维码）。
5. 申报主体2025年度获得天使投资、风险投资和私募股权投资的证明材料。
6. 申报主体2025年度获得投资总额的收款凭证。
7. 其他需要补充的有关证明材料。

**政策咨询：** 市经信局人工智能产业处，电话：028-61884670
