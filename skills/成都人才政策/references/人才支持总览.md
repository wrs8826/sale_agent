# 成都人才政策总览

## 一、蓉漂计划（成都市级，引进类主干计划）

**定位：** 成都市面向国内外高层次人才的主力引进计划，覆盖顶尖、领军、青年等多个层次  
**适用人群：** 2019年1月1日及以后引进到成都市的高层次人才及团队  
**申报截止（2024年批次）：** 2024年7月31日

| 项目 | 年龄上限 | 核心门槛 |
|---|---|---|
| 顶尖人才 | — | 战略科学家/两院院士，依托高能级创新平台 |
| 创新领军人才 | 65周岁 | 知名高校/科研机构正高级职务或相当层次 |
| 创业领军人才 | 65周岁 | 来蓉创办科技型企业，货币出资≥100万元 |
| 创新青年人才 | 40周岁（优秀42岁） | 在蓉从事应用研究或技术开发 |
| 创业青年人才 | 40周岁（优秀42岁） | 来蓉创办科技型企业，货币出资≥50万元 |
| 创新创业团队 | — | 带头人+4名成员，研发投入≥200万元 |

## 二、天府峨眉计划（四川省级计划，成都区县组织申报）

**定位：** 四川省级高层次人才引进计划，通过成都各区县渠道申报，分前置改革项目和常规项目  
**申报截止（2025年高新区批次）：** 2025年9月23日 17:00  
**申报系统：** 四川人才工作网（http://www.scrcgz.com/），使用2025版申报软件

| 类别 | 项目 | 年龄上限 |
|---|---|---|
| 前置改革 | 顶尖人才 | 70周岁 |
| 前置改革 | 创新领军人才 | 65周岁 |
| 前置改革 | 国防科技领域人才专项 | 60周岁 |
| 常规·个人 | 创业领军人才 | 65周岁 |
| 常规·个人 | 创新领军人才 | 65周岁 |
| 常规·个人 | 青年人才 | 40周岁 |
| 常规·专项 | 国防科技、省校省院省企、乡村振兴、**重点产业**（新） | ≤60周岁 |
| 常规·团队 | 创业团队、创新团队 | — |

**2025年相较2024年的主要变化：**
- 合同及实缴截止日期整体后移1年（合同签订截止 2025-08-31，实缴截止 2025-08-31）
- 专项引进新增**重点产业人才专项**，取代原"绿色低碳优势产业人才专项"；专项引进年龄明确不超过60周岁
- 新增**非共识人才通道**：创新领军和青年人才可由两院院士/国家级人才举荐申报，不作学历职称要求
- 创业领军学历放宽条件改为"拥有广阔市场前景的自主创新产品或已获风险投资机构投资"（可放宽至学士）
- 破格申报比例明确不超过本单位推荐总人数的 **40%**
- 青年人才/专项入选后转项须"在川满3年且来川6年内"方可转为创业领军

## 三、卧龙九条——AI产业人才生态（成都市级）

**定位：** 专项支持人工智能产业四类人才梯队（顶尖→领军→应用→高潜）

| 人才类型 | 核心支持 |
|---|---|
| 顶尖人才 | 最高 **10 亿元**综合支持 |
| 领军人才 | 市级人才计划、双聘机制、"蓉城AI合伙人计划" |
| 应用人才 | 订单式培养、工匠培育、来蓉创业支持 |
| 高潜人才 | 高潜人才库、博士后专项、"蓉漂杯"赛道 |

全链条奖励：
- "揭榜挂帅"项目：最高 1000 万元
- "爆款"产品：最高 100 万元
- "标杆应用场景"：最高 100 万元
- 纯信用贷款：最高 1000 万元

## 计划层级关系说明

```
四川省级：天府峨眉计划（通过区县渠道申报）
     ↓
成都市级：蓉漂计划（市委组织部主管）
          卧龙九条（AI产业专项，市经信局）
          AI产业专项补贴（市经信局）
     ↓
区级：成都高新区高层次人才创业支持（高新区）
      科技人才创新创业资助（市科技局，2020）
```

- 已入选"天府峨眉计划"的，不得申报"天府青城计划"
- 已入选"蓉城英才计划"的，不得申报"蓉漂计划"
- 经成都市渠道推荐入选国家海外高层次人才引进计划的，直接纳入"蓉漂计划"对应项目支持
