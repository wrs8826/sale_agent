# 2025年"天府峨眉计划"——注意事项与材料报送

**申报截止（材料报送）：** 2025年9月23日 17:00

## 一、注意事项

1. **申报方式：** 线上填报 + 线下报送纸质材料，申报软件于"四川人才工作网"（http://www.scrcgz.com/）下载（**使用2025版**）。
2. **特殊申报：** 特别优秀且为四川特需的前置改革项目可"一事一议"；常规项目突破学历、职称、工作经历、任职年限等限制须由用人单位提交书面破格申请，**破格人数占本单位推荐总人数的比例原则上不超过40%**。
3. **成果填报：** 常规项目每类成果（项目/论文/专利/产品/获奖）一般不超过5项。
4. **尽职调查：** 用人单位需对申报人知识产权、竞业禁止、保密约定等进行综合评估，在申报书"用人单位推荐意见"栏逐项填写核实意见。
5. **申报限制：**
   - 每人每年只能申报一个项目，前置改革项目和常规项目不得同期申报
   - 青年人才和专项引进项目入选后，须在川发展满3年且来川6年内，符合条件的**仅可申报转为创业领军人才项目**
   - 已入选"天府青城计划"的不得申报"天府峨眉计划"
   - 已入选国家级人才计划且在支持期内的，可申报但不重复授予人才称号
6. **资金使用：** 资助资金采取"当年立项、次年支出"方式发放。

## 二、材料报送要求

**报送截止：** 2025年9月23日 17:00，逾期不予受理，不接受个人申报。

**电子版（U盘）：** 申报书扫描件、附件材料扫描件、申报数据压缩包（系统导出，勿进行任何操作）、申报人选/团队情况汇总表Excel版。文件夹命名格式：`项目类型-单位名称-申报人姓名`。

**纸质版：** 申报书一式8份、附件材料一式2份（封面封底为白色A4纸双面印刷胶装，附件材料需注明页码和目录，汇总表加盖公章装订于附件封皮后、目录前）。
