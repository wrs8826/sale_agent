# 2026年成都AI产业专项补贴——申报方式与通用要求

**文号：** 成经信财〔2026〕4号  
**申报截止：** 企业2026年4月30日23:59；区县初审2026年5月15日23:59  
**申报方式：** 线下申报，材料由企业报送至企业所属区（市）县工业和信息化、新经济主管部门。

## 补贴项目总览

本通知包含4类补贴项目：
1. **模型使用补贴**（最高100万元）— 按年度模型使用费用30%补贴
2. **标志性应用场景奖励**（最高100万元）— 按场景建设实际投入30%奖励
3. **企业融资奖励**（最高100万元）— 按实际获得投资额10%奖励
4. **参展补贴**（最高10万元/展）— 按展位费+展品运输费60%补贴

## 通用要求

1. 申报单位对申报资料的真实、完整和准确性负责；所有申报材料须真实有效，提供的复印件均须加盖申报单位公章，全套材料加盖骑缝章。
2. 市级各专项资金对同一项目原则上不重复支持；同一企业同一项目补贴按就高不就低原则只能享受一次。
3. 申报了中央、省补助资金或多项市级专项资金的，需在申报材料中说明已获得或正在申报的补助资金情况。
4. 市经信局未委托任何机构或个人代理申报事宜，自主申报，不收取任何费用。
5. 补贴（奖励）金额低于10万元（不含）不纳入补贴范围（中小企业除外，中小企业需提供《中小企业声明函》）。
6. "人工智能业务收入"是指开展人工智能软件开发、人工智能硬件与设备制造、人工智能服务的相关收入。
7. 本政策认定的金额不含税。
